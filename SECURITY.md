# Security Policy for SchemaGlow

## Supported Versions

Only the current major version is actively supported for security updates.

| Version | Supported |
| ------- | --------- |
| v0.x.x  | Yes |

## Reporting a Vulnerability

Do not report vulnerabilities in public issues.

Email: `workwithgoutam@gmail.com`

Maintainer response target:

1. Acknowledge receipt within 48 hours.
2. Confirm severity and remediation plan.
3. Publish a fix and advisory once a patch is available.
