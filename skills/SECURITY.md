# Security Policy for SchemaGlow

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v0.x.x  | Yes |

## Reporting

Report vulnerabilities privately to `workwithgoutam@gmail.com`.

Target response:

1. Acknowledge within 48 hours.
2. Confirm remediation plan.
3. Publish a fix and advisory after release.
